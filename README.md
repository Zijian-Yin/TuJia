## 工程描述：

> 核心技术栈：Vue3、Vite、Pinia、Vant、Less、Axios

> 由于其他页面大同小异，故只开发了最有挑战性的主页

> 数据是动态向服务器发送请求获取的，对Axios库进行了二次封装，这是考虑到工程的健壮性
